Nether Fights

This game was the end result of one of my 2nd year courses, C++ Programming. It was my first introduction to C++, so it was quite the task to make a working 2D fighter game (the given theme). Because I wanted it to be a presentable portfolio piece, I decided, when starting with the assignment, to give the theme a twist.

At the time, I was playing (& quite obsessed with) the new DOOM Eternal, hence the DOOM theme.
